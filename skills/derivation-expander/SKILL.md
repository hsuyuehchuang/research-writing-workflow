# Derivation Expander

## Purpose

Provide extremely detailed expansions only when explicitly requested.

## Responsibilities

- expand one selected section at a time
- place expanded material in an appendix
- preserve connection to the concise main-line derivation

## Rules

- Never expand the entire note by default.
- Make every expansion traceable back to the main text.
- Keep expansions local, not sprawling.
