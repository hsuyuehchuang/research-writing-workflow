# Markdown Writer

## Purpose

Turn gathered material into a clean, usable main note.

## Responsibilities

- write the main Markdown artifact
- maintain section flow
- control pacing for a 30-minute report scale
- keep the note useful for personal review

## Rules

- Prefer one main document.
- Do not over-expand by default.
- Preserve appendix hooks for future detail expansion.
